## ✨ Features

- ⚡ **Zero dependencies** — pure HTML, CSS, JavaScript
- 🎨 **Custom color palette** — Turmeric `#FFBE0B` & Malt `#2A2313`
- 📱 **Fully responsive** — works on mobile, tablet, and desktop
- 🌙 **Dark theme** — professional dark background with golden accents
- 🔢 **Scroll animations** — fade-up reveal on scroll using IntersectionObserver
- 🧭 **Fixed navigation** — shrinks on scroll, hamburger menu on mobile
- 🖋️ **Google Fonts** — Playfair Display, Inter, Space Mono
- 🗂️ **Separated files** — clean `index.html`, `style.css`, and `script.js`

## 🚀 Getting Started

### Option 1 — Open Directly (Quickest)
Just double-click `index.html` — it opens in your browser instantly.

### Option 2 — VS Code Live Server (Recommended)
1. Open the project folder in **VS Code**
2. Install the **Live Server** extension by Ritwick Dey
3. Right-click `index.html` → **Open with Live Server**
4. Site runs at `http://127.0.0.1:5500`

> ⚠️ **Important:** All 3 files (`index.html`, `style.css`, `script.js`) must be in the **same folder**.
